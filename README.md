# Проекты

## Медицинский VR-симулятор «Виртуальный пациент»
Симулятор, обучающий навыкам работы врача-терапевта. Разработан в Unity, для создания 3D-контента использовались также Unreal Engine и Blender. Существует в двух видах - обычный для Wimdows и VR. Версия VR работает через соединение с компьютером и очками PICO. В программе реализован встроенный редактор сценариев, режим экзамена и защита от копирования. Всю работу проделал один, за исключением консультации врача, по вопросом медицины. Разработка заняла примерно 1 год. У персонажей реализована мимика, они могут говорить, речь генерируется по тексту.
